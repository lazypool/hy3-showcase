下面是一个**简单易懂的 Python 快速排序（Quick Sort）实现**，使用经典的“分治法”思路：

```python
def quick_sort(arr):
    # 基线条件：空数组或只有一个元素时直接返回
    if len(arr) <= 1:
        return arr

    # 选择基准值（这里选第一个元素）
    pivot = arr[0]

    # 分区
    left = [x for x in arr[1:] if x < pivot]
    right = [x for x in arr[1:] if x >= pivot]

    # 递归排序并合并
    return quick_sort(left) + [pivot] + quick_sort(right)


# 示例
if __name__ == "__main__":
    data = [5, 2, 9, 1, 5, 6]
    print(quick_sort(data))
```

### 说明
- **时间复杂度**
  - 平均：`O(n log n)`
  - 最坏（已排序且选第一个为基准）：`O(n²)`
- **空间复杂度**：`O(n)`（因为新建了列表）
- 适合**教学和理解原理**
